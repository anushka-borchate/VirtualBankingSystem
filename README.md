# 🏦 Virtual Banking System

A desktop-based Virtual Banking System built using **Java Swing (GUI)**.

## 📌 Features
- 🔐 Admin Login & Dashboard (`Alogin`, `Adashboard`)
- 👤 Employee Login (`Elogin`)
- 🆕 New User Registration (`Nlogin`)
- 🏠 User Home Page (`Home`)
- 💰 Deposit Money (`Deposit`)
- 💸 Withdraw Money (`Withdraw`)
- 🔄 Fund Transfer (`Transfer`)
- 📒 Passbook / Transaction History (`Passbook`)
- 👤 User Profile (`Profile`)
- 🧠 Quiz Feature (`QuizApp`, `Questions`)
- 🛬 Landing Page (`Landing`)

## 🛠️ Tech Stack
- **Language:** Java
- **GUI Framework:** Java Swing (JFrame)
- **IDE:** IntelliJ IDEA
